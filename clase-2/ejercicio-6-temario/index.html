Curso de desarrollo Frontend

Lo primero que tenemos que saber sobre HTML es que no es un leguaje de programación, es un lenguaje de marcado. La sigla corresponde a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto. Un lenguaje de marcado se compone de etiquetas jerárquicas que permiten mostrar contenido en una página web, como textos, imágenes, videos, etc. Podemos decir que el HTML describe la estructura de nuestro documento o página utilizando etiquetas (tags).
